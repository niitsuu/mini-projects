# Login Page

A modern and responsive login page built with HTML and CSS.

## Preview

![Login Page Preview](./preview.jpg)

## Features

- Username and password input fields
- Remember me checkbox
- Forgot password link
- Register link
- Modern glassmorphism design
- Responsive centered layout

## Technologies Used

- HTML
- CSS
- Boxicons
- Flexbox

## Purpose

This project was created to practice front-end structure, styling, alignment, and user interface design.

## What I practiced

- Form layout structure
- Input styling
- Icon positioning
- Hover effects
- Backdrop blur (glassmorphism)
- Centering elements with **Flexbox**
